# BM10 两个链表的第一个公共结点

## Solution

```javascript
/*function ListNode(x){

function FindFirstCommonNode(pHead1, pHead2) {
    // write code here
    let p1 = pHead1,
        p2 = pHead2;
    while (p1 !== p2) {
        p1 = !p1 ? pHead2 : p1.next;
        p2 = !p2 ? pHead1 : p2.next;
    }
    return p1;
}
module.exports = {
    FindFirstCommonNode: FindFirstCommonNode,
};
```
