# LC-0926. Flip String to Monotone Increasing

A binary string is monotone increasing if it consists of some number of `0`'s (possibly none), followed by some number of `1`'s (also possibly none).

You are given a binary string `s`. You can flip `s[i]` changing it from `0` to `1` or from `1` to `0`.

Return _the minimum number of flips to make `s` monotone increasing_.

Example 1:

```
Input: s = "00110"
Output: 1
Explanation: We flip the last digit to get 00111.
```

Example 2:

```
Input: s = "010110"
Output: 2
Explanation: We flip to get 011111, or alternatively 000111.
```

Example 3:

```
Input: s = "00011000"
Output: 2
Explanation: We flip to get 00000000.
```

Constraints:

-   `1 <= s.length <= 10^5`
-   `s[i]` is either `'0'` or `'1'`.

## Solution

### 动态规划

`s[i]` 为 `'1'` 时不影响翻转，否则取反转当前和翻转 `s[0 : i - 1]` 中所有 `1` 的次数的最小值。`count` 记录这部分中的 `1` 的个数。

```javascript
/**
 * @param {string} s
 * @return {number}
 */
var minFlipsMonoIncr = function (s) {
    let cur = 0,
        count = 0;
    // 因为要统计1的个数，从0开始遍历
    for (let i = 0; i < s.length; i++) {
        if (s.charAt(i) == "1") count++;
        else cur = Math.min(cur + 1, count);
    }
    return cur;
};
```
