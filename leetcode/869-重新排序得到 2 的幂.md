# LC-0869. Reordered Power of 2

You are given an integer `n`. We reorder the digits in any order (including the original order) such that the leading digit is not zero.

Return `true` _if and only if we can do this so that the resulting number is a power of two_.

**Example 1:**

```
Input: n = 1
Output: true
```

**Example 2:**

```
Input: n = 10
Output: false
```

**Example 3:**

```
Input: n = 16
Output: true
```

**Example 4:**

```
Input: n = 24
Output: false
```

**Example 5:**

```
Input: n = 46
Output: true
```

**Constraints:**

-   `1 <= n <= 10^9`

## Solution

### 回溯

```javascript
/**
 * @param {number} n
 * @return {boolean}
 */
var reorderedPowerOf2 = function (n) {
    let nums = (n + "").split("");
    let len = nums.length;
    let temp = [];
    let flag = false;
    const visited = new Array(len).fill(0);
    const backtrack = () => {
        if (temp.length == len) {
            if (temp[0] != "0") {
                // 不能有前导零
                let num = Math.log2(parseInt(temp.join("")));
                if (num == Math.floor(num)) flag = true;
            }
            return;
        }
        for (let i = 0; i < nums.length; i++) {
            if (visited[i] === 1) continue;
            visited[i] = 1;
            temp.push(nums[i]);
            backtrack();
            temp.pop();
            visited[i] = 0;
        }
    };

    backtrack();
    return flag;
};
```
